校园智能交通违规检测系统 (V2.0)

📝 项目简介

本项目是一款专门针对校园场景开发的智能交通监控系统。系统集成了深度学习目标检测、多目标追踪、字符识别及轨迹预测算法，实现了从视频采集到违规判定的全流程自动化管理。


🚀 核心技术架构

系统采用了多模型融合方案，构建了 感知-追踪-识别-预测 的闭环体系：

目标检测 (YOLOv8)：实时识别校园内的车辆（汽车、大巴、卡车）及行人，精准定位。

多目标追踪 (IoU Tracker)：基于交并比算法为每个目标分配唯一 ID，实现跨帧轨迹关联。

车牌属性识别 (CRNN)：利用卷积循环神经网络对车辆局部特征进行提取，识别车牌号码、颜色及细分车型。

行为轨迹预测 (LSTM)：通过长短期记忆网络分析历史坐标，预测未来 3 秒行驶轨迹，并自动判定车辆逆行行为。


✨ 主要功能

违规禁行检测：自动判定车辆是否进入宿舍区等禁行区域。

违规禁停检测：基于追踪轨迹判断车辆在禁停区的滞留时间。

轨迹与逆行判定：实时预测车辆走向，针对逆行行为自动触发告警。

车辆身份识别：自动记录违规车辆的车牌号与车型，方便事后追溯。

可视化管理后台：提供任务管理、违规事件列表、结果视频下载及统计分析报表。


🛠️ 快速开始

1. 环境准备
Bash
pip install -r requirements.txt
2. 运行检测逻辑
Bash
python detect_violation.py --weights yolov8n.pt --source "your_video.mp4" --show
3. 启动 Web 管理系统
Bash
python run.py
访问 http://127.0.0.1:5000 即可进入管理后台。


📂 项目结构

webapp/：Flask 后端逻辑及前端模板。

configs/：违规判定规则（Polygon 坐标）配置文件。

runs/：检测结果视频及 CSV 事件记录存储。

models/：预训练模型权重存储。
